This is the Scala implementation of the quest adaptation algorithm by Boyang Li and Mark O. Riedl. The algorithm is described in 

Boyang Li and Mark O. Riedl. Creating Customized Game Experiences by Leveraging Human Creative Effort: A Planning Approach. In Agents for Games and Simulations II. Frank Dignum (Ed.) LNAI 6525. 2011. pp. 99-116. 

The original program was implemented in Lisp. This is a port to Scala. It's current status is incomplete.
